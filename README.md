# Репозиторий backend по курсу "Разработка интернет-приложений"

Расчет энергопотребления системы видеонаблюдения

Разработанное приложение позволит исследователям создавать и формировать заявки на расчёты энергопотребления системы видеонаблюдения для выбранных ими камер, а инженерам завершать или отклонять данные заявки.
